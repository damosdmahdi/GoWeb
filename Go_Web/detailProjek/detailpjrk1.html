Sistem Informasi dan Manajemen Data UMKM Terintegrasi

Sistem ini merupakan sebuah platform berbasis web yang dirancang untuk menjadi pusat data utama bagi Usaha Mikro, Kecil, dan Menengah (UMKM) di suatu wilayah regional. Aplikasi ini berfungsi untuk melakukan pendataan, pengelolaan, serta visualisasi informasi UMKM secara terpusat dan digital. Tujuannya adalah untuk menggantikan metode pendataan manual yang tidak efisien dan rentan terhadap kesalahan, serta menyediakan data yang akurat dan real-time bagi pemerintah daerah untuk mendukung pengambilan keputusan strategis. Selain itu, sistem ini juga menyajikan dashboard publik yang informatif untuk masyarakat umum.

Latar Belakang / Tujuan Proyek

  • Masalah yang ingin diselesaikan:
    Pemerintah daerah seringkali menghadapi kesulitan dalam mengelola data UMKM yang tersebar, tidak terstandarisasi, dan masih bersifat manual (menggunakan spreadsheet atau dokumen fisik). Hal ini menyebabkan data tidak akurat, sulit diakses, dan lambat dalam proses rekapitulasi, sehingga perencanaan program bantuan atau pelatihan menjadi tidak optimal.
  • Peluang atau alasan mengapa proyek ini penting:
    Dengan adanya sistem terpusat, pemerintah dapat memiliki "sumber kebenaran tunggal" (single source of truth) untuk data UMKM. Ini membuka peluang untuk membuat kebijakan yang lebih tepat sasaran, meningkatkan transparansi data kepada publik, dan mempercepat proses pelayanan serta verifikasi data UMKM di wilayah tersebut.
  • Hasil yang ingin dicapai (output):
    Output dari proyek ini adalah sebuah aplikasi web fungsional yang memiliki dua bagian utama:
    1. Antarmuka Publik (Frontend): Sebuah dashboard informatif yang menampilkan statistik kunci secara dinamis, seperti total pertumbuhan UMKM per tahun dan kontribusi UMKM per wilayah dalam bentuk grafik interaktif.
    2. Panel Admin (Backend): Sebuah sistem di belakang layar yang dilindungi otentikasi untuk petugas yang berwenang melakukan manajemen data (CRUD - Create, Read, Update, Delete) pada data UMKM, data wilayah, dan data statistik.

Spesifikasi / Kebutuhan Teknis

  • Fitur, fungsi, atau persyaratan yang harus dipenuhi:
    • Dashboard Publik: Menampilkan ringkasan data dan statistik vital kepada pengunjung.
    • Visualisasi Data Dinamis: Menampilkan data dalam bentuk Grafik Batang (pertumbuhan per tahun) dan Grafik Lingkaran (kontribusi per wilayah) yang datanya diambil langsung dari database.
    • Manajemen Data UMKM (CRUD): Fungsi bagi admin untuk menambah, melihat, mengedit, dan menghapus data UMKM.
    • Manajemen Data Pendukung: Kemampuan untuk mengelola data master seperti data wilayah dan data statistik pertumbuhan.
    • Manajemen Konten Halaman Depan: Kemampuan admin untuk mengubah konten dinamis seperti daftar fitur dan info kontak.
    • Sistem Otentikasi: Halaman login untuk membatasi akses ke panel admin hanya untuk pengguna yang berwenang.
  • Standar atau teknologi yang digunakan:
    • Frontend: HTML5, CSS3, dan JavaScript untuk membangun struktur dan interaktivitas halaman.
    • Styling: Menggunakan framework CSS Bootstrap 5 untuk mempercepat pengembangan dan memastikan desain yang konsisten dan responsif.
    • Backend: Dibangun menggunakan bahasa pemrograman PHP 8.2.
    • Database: Menggunakan database relasional MySQL (MariaDB) yang dikelola melalui phpMyAdmin.
    • Lingkungan Pengembangan: XAMPP pada sistem operasi Linux (Ubuntu).
